# Compiler Construction: Theory and Implementation

This repository tracks my progress through the **Compiler Construction I & II** courses. The goal is to bridge the gap between theoretical formal languages and the practical engineering of a fully functional compiler.

## Project Overview

The project is structured into two main academic milestones:

### 1. Compilers I: Fundamentals & Core Phases 
Focused on the "under the hood" mechanics of translation. In this stage, I am implementing the individual components of the compiler's *front-end*:
- **Lexical Analysis (Scanner):** Tokenizing the source stream using DFAs and Regular Expressions.
- **Syntax Analysis (Parser):** Validating grammatical structure (Context-Free Grammars, LL/LR Parsing).
- **Semantic Analysis:** Type checking, scope validation, and logical consistency.
- **Symbol Table:** Efficient management of identifiers and metadata.
